# Một lưu ý quan trọng trong `hybrid_search`

Bước cuối cùng là:

> map ngược id → `Chunk`

Đừng duyệt toàn bộ danh sách `chunks` mỗi lần tìm kiếm:

python
# Hoạt động nhưng O(n)
for chunk in bm25_index.chunks:
    if chunk.chunk_id == target_id:
        ...


Thay vào đó, khi khởi tạo `BM25Index`, bạn nên tạo sẵn một dictionary:

python
self.chunk_map = {
    chunk.chunk_id: chunk
    for chunk in chunks
}


Sau đó chỉ cần:

python
result_chunks = [
    bm25_index.chunk_map[chunk_id]
    for chunk_id in top_ids
]


Tra cứu dictionary có độ phức tạp trung bình **O(1)**, nhanh hơn rất nhiều nếu sau này bạn có hàng chục nghìn chunk. Đây cũng là cách các hệ thống retrieval thực tế thường triển khai.
